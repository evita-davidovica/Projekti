# Laikapstāļu lietotne

CLI rīks ar Open-Meteo API, datu saglabāšanu un statistiku.

## Uzstādīšana
1. `npm install` (pēc izvēles)
2. `npm start vai node index.js`


## Lietošana
1. Atver termināli
2. `npm run test-api` - tests darbojas
3. `npm start` - galvenā programma

## Piemēri
1. Iegūt Rīgas laikapstākļus
2. Skatīt vēsturi
3. Iziet

## Failu struktūra
- `weather-data.json` - galvenie dati
- `api.js` - API integrācija un testi
- `index.js` - CLI izvēlne
